Welcome to the RedSec learning platform — a free, open, GitBook-style guide to
offensive security. No accounts, no paywalls. Read top to bottom, or jump to
whatever you need from the sidebar.

## How this is organized

The docs are split into sections you can work through in any order:

- **Getting started** — what red teaming is and how to set up.
- **CTFs** — capture-the-flag formats, tooling, and worked challenges.
- **Bug bounties** — finding and reporting real-world vulnerabilities for reward.
- **Finding vulnerabilities** — the core bug classes and how to hunt them.

Each section is a stack of short pages. Use the sidebar to move between them and
the "on this page" list to jump within a page.

<Callout type="warning" title="Authorized use only">
  Everything here is for **authorized testing and education**. The techniques
  are real and often illegal to use without permission. Only ever apply them to
  systems you own or have explicit, written authorization to test.
</Callout>

## Where to start

If you're new, continue to [your environment](/getting-started/environment) to
get a working setup, then pick the section that matches your goal.
